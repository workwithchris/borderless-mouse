Quick start

tar xzf borderless-mouse-linux-x86_64.tar.gz
sudo cp borderless-mouse /usr/local/bin/

On the server (Linux):

borderless-mouse server --bind 0.0.0.0

On the client (Mac):

borderless-mouse client --connect <linux-ip>

Features

  • Seamless mouse movement across screens
  • Keyboard & click forwarding
  • Bi-directional clipboard sync
  • Desktop GUI for easy control
  • Headless CLI mode
  • Direct Ethernet link setup
  • No cloud, no subscription
  • Pure Rust — no runtime deps
